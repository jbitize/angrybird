# 갤럭시 게임 만들기 - 단계별 프롬프트

아빠가 클로드한테 이렇게 말하면 게임이 만들어져. 하나씩 따라해봐!

---

## 1단계: 기본 게임

```
아이폰 사파리에서 돌아가는 우주 슈팅 게임 만들어줘. index.html 하나로.
화면 아래쪽에 우주선이 있고, 손가락으로 좌우로 움직여.
우주선은 자동으로 총알을 쏘고, 위에서 적들이 내려와.
적을 맞추면 터지고 점수 올라가. 적한테 부딪히면 게임오버.
배경은 까만 우주에 별이 반짝이는 느낌으로.
```

## 2단계: GitHub에 올려서 폰으로 플레이

```
지금 만든 거 git에 커밋하고 push 해줘.
GitHub Pages로 배포해서 아이폰에서 URL로 바로 플레이할 수 있게.
```

## 3단계: 파워업 아이템

```
적을 죽이면 가끔 아이템이 떨어지게 해줘.
빨간 아이템 먹으면 총알이 3발로 퍼져나가고,
파란 아이템 먹으면 보호막이 1번 생겨.
```

## 4단계: 보스

```
적 30마리 죽이면 보스가 나오게 해줘.
보스는 크고 체력바가 있어. 총알도 쏴.
보스 죽이면 "STAGE CLEAR" 뜨고 다음 스테이지로.
스테이지 올라갈수록 적이 빨라지고 많아져.
```

## 5단계: 이펙트 추가

```
총알 맞으면 폭발 이펙트 넣어줘. 파티클로.
우주선 움직일 때 엔진 불꽃도 나오게.
보스 죽을 때는 큰 폭발로. 화면이 잠깐 흔들리면 좋겠어.
```

## 6단계: 사운드

```
효과음 넣어줘. Web Audio API로.
총 쏠 때 삐융, 적 터질 때 펑, 아이템 먹을 때 띠링.
보스전에는 긴박한 BGM도 넣어줘.
전부 코드로 생성해서 파일 없이.
```

## 7단계: 꾸미기

```
우주선이랑 적 디자인 좀 멋있게 바꿔줘.
우주선은 날개 달린 전투기 느낌, 적은 외계인 느낌으로.
보스는 좀 무섭게. 전부 Canvas로 그려줘.
점수 폰트도 게임 느낌 나게.
```
