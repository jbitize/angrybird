# Galaxy Shooter - 대규모 콘텐츠 업데이트

## 1. 콤보 시스템
- 연속 킬 시 콤보 카운트 증가 (2초 내 다음 킬)
- 콤보 배율: x2, x3, x4... 점수에 곱셈 적용
- 화면에 "COMBO x5!" 글로우 텍스트 표시
- 콤보 끊기면 리셋

## 2. 적 무기 강탈
- Shooter 적이 가끔 보라색 "강탈 총알" 발사
- 맞으면 현재 무기가 normal로 리셋됨
- 강탈 총알은 시각적으로 구별 (보라색 + 물결)

## 3. 소행성 장애물
- 랜덤으로 소행성 등장 (큰 회색 돌)
- 총알로 부수면 높은 확률로 아이템 드랍 (50%)
- 부수지 못하면 충돌 시 데미지
- HP 3~5발, 맞을수록 금 가며 부서짐

## 4. 보스 타입 다양화 (스테이지별 로테이션)
- **스테이지 1, 4, 7...**: 기존 외계 전함 (3페이즈)
- **스테이지 2, 5, 8...**: 분열 보스 - 처치 시 소형 2마리로 분열
- **스테이지 3, 6, 9...**: 촉수 보스 - 촉수 휘두르기 + 넓은 범위 공격

## 5. 목숨 시스템
- 시작 시 3목숨 (하트 표시)
- 죽으면 목숨 -1, 잠시 무적 후 리스폰
- 0목숨에서 죽으면 진짜 게임오버

## 6. 하이스코어
- localStorage에 최고 점수 저장
- 게임오버 시 "NEW RECORD!" 표시
- HUD에 하이스코어 상시 표시
