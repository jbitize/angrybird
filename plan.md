# Galaxy Shooter - 화끈한 난이도 & 무기 업데이트

## 1. 난이도 대폭 상향
- 초기 스폰 간격: 40 → 25프레임 (적이 훨씬 많이 나옴)
- 초기 적 속도: 1.5배로 시작
- 스테이지당 속도 배율: +30% → +40%
- 동시에 2~3마리씩 스폰되는 웨이브 패턴 추가

## 2. 새로운 무기 아이템 (6종)
- **빨강 (Triple Shot)**: 3발 부채꼴 (기존)
- **주황 (Rapid Fire)**: 발사 속도 3배 빨라짐
- **노랑 (Laser Beam)**: 화면 위까지 관통하는 레이저빔
- **초록 (Homing Missile)**: 적을 자동 추적하는 미사일 2발
- **파랑 (Shield)**: 1회 방어 (기존)
- **보라 (Bomb)**: 화면 전체 적 폭발 (화면 클리어)

## 3. 아이템 드랍률 조정
- 드랍률: 10% → 18% (무기가 많으니 자주 줘야 재미있음)
- 무기 지속시간: 4초 (Bomb/Homing은 즉시 효과)

## 4. 무기 비주얼
- 각 무기별 고유 총알 색상/이펙트
- 레이저: 굵은 빔 + 글로우
- 호밍 미사일: 꼬리 연기 파티클
- 폭탄: 화면 전체 플래시 + 대규모 폭발
