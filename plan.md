# Galaxy Shooter - 대규모 업데이트

## 1. 스테이지 & 보스 시스템
- 적 30마리 처치 → 보스 등장
- 보스: 큰 크기, 체력바 표시, 총알 발사 (플레이어 방향으로)
- 보스 처치 → "STAGE CLEAR" 표시 → 3초 후 다음 스테이지
- 스테이지 올라갈수록: 적 스폰 빠르게 + 적 속도 증가

## 2. 이펙트
- **총알 폭발**: 적 피격 시 파티클 이펙트 (기존 폭발 강화)
- **엔진 불꽃**: 우주선 이동 시 꼬리 파티클 (주황/노랑)
- **보스 폭발**: 대형 폭발 + 화면 흔들림 (screen shake)

## 3. 사운드 (Web Audio API, 코드 생성)
- 총 발사: 삐융 (고주파 sweep down)
- 적 폭발: 펑 (노이즈 burst)
- 아이템 획득: 띠링 (2음 상승)
- 보스전 BGM: 긴박한 루프 (저음 펄스 + 리듬)

## 4. 비주얼 개선 (전부 Canvas)
- **우주선**: 날개 달린 전투기 스타일 (더 디테일하게)
- **일반 적**: 외계인 느낌 (촉수/안테나)
- **보스**: 크고 무서운 외계 전함 느낌
- **점수 폰트**: 게임 느낌 나는 스타일링 (그림자, 글로우)

## 5. UI
- 스테이지 번호 표시
- 보스 체력바 (화면 상단)
- 킬 카운트 표시 (30마리 중 몇 마리)
